### 4. 系统使用技术

前端：html+scss+js+es6+vue+element+vuex+mock.js(没有数据时模拟)+echarts

后端：java+Springboot+mybatis+多数据源+redis 数据库+连接池+yml配置方式+swagger

数据库：mysql

数据来源：东方财富网

爬取数据技术：python+scapy 框架
